## 简介
AOS是一个运行在x86-64上，由UEFI引导的操作系统。AOS是该项目当前暂定代号，待项目成熟时,会新建仓库并在该文档中添加链接。  
该项目当前以MIT协议开源。

当前有以下模块项目：
- **/uefi/AOS** [🟡]   
EDK2项目，模块命名为`aos.uefi`。内部分为模块主体和一个命名为`aos.uefi.timer`的EDK2库。该模块作为操作系统的UEFI引导程序存在，提供了设置CPU功能、获取重要表、获取图形输出信息、为内核文件进行安全校验并提供随机化基址、加载内核文件等重要功能。目前还在完善，未有第一版。
- **/kernel** [🟡]  
由CMake+Ninja支持构建的内核文件项目，模块命名为`aos.kernel`。其负责在接受引导程序传递来的启动参数后，构造内核运行环境并提供相关的核心内核服务。在模块`aos.uefi`引导下，其程序文件将存放于ESP下`/aos`目录下，不会也禁止将ESP暴露于系统中。
